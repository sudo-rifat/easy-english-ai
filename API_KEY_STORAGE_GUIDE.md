## â“ FAQ

### Q: Is my key really secure?
A: **Yes!** It's stored locally on your device, not on any server. Even we can't see it.

### Q: What if someone steals my laptop?
A: They'll have access to your key. Immediately delete it from Settings if this happens.

### Q: Can I use same key on different devices?
A: **Yes!** You'll need to enter the same key on each device. Each device has its own LocalStorage.

### Q: What if I change providers?
A: New provider automatically saves when you analyze. Old provider is replaced.

### Q: Does it work on mobile?
A: **Yes!** Mobile browsers also have LocalStorage. Works on any device.

### Q: Can I share my saved key with friends?
A: **No!** Keys are per-device. They would need to enter their own key. (Which is safer anyway!)

### Q: What if I clear my browser cache?
A: Your key will be deleted. You'll need to enter it again.

### Q: How long does the key stay saved?
A: **Forever** - until you:
- Manually delete it
- Clear browser data
- Uninstall the browser
- Change browser

### Q: Is there a limit to how long the key stays?
A: **No time limit!** The key stays until you delete it.
